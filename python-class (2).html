¿Qué es Python?

Un lenguaje de programación poderoso y versátil

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Creado por Guido van Rossum en 1991, se ha convertido en uno de los lenguajes más populares del mundo.

Características principales

  • Sintaxis clara y legible: Python se lee casi como inglés, lo que facilita su aprendizaje
  • Interpretado: No necesitas compilar, ejecutas directamente el código
  • Multiplataforma: Funciona en Windows, Mac, Linux y más
  • Amplia biblioteca estándar: Miles de módulos listos para usar
  • Multiparadigma: Soporta programación procedural, orientada a objetos y funcional
💡 Dato curioso: El nombre "Python" viene de los Monty Python, el grupo de comedia británico, ¡no de la serpiente!

¿Dónde se usa Python?

🤖 Inteligencia Artificial

Machine Learning, Deep Learning, procesamiento de lenguaje natural

🌐 Desarrollo Web

Django, Flask, FastAPI para crear aplicaciones web potentes

📊 Análisis de Datos

Pandas, NumPy, Matplotlib para ciencia de datos

⚙️ Automatización

Scripts para automatizar tareas repetitivas

El Zen de Python

La filosofía de Python

El Zen de Python es un conjunto de 19 principios que guían el diseño y uso del lenguaje Python. Fue escrito por Tim Peters en 1999 y se ha convertido en la filosofía fundamental de la comunidad Python.

💡 Easter Egg: Puedes ver el Zen de Python escribiendo import this en el intérprete de Python.

Los 19 principios

The Zen of Python, by Tim Peters 
Beautiful is better than ugly.
Explicit is better than implicit.
Simple is better than complex.
Complex is better than complicated.
Flat is better than nested.
Sparse is better than dense.
Readability counts.
Special cases aren't special enough to break the rules.
Although practicality beats purity.
Errors should never pass silently.
Unless explicitly silenced.
In the face of ambiguity, refuse the temptation to guess.
There should be one-- and preferably only one --obvious way to do it.
Although that way may not be obvious at first unless you're Dutch.
Now is better than never.
Although never is often better than *right* now.
If the implementation is hard to explain, it's a bad idea.
If the implementation is easy to explain, it may be a good idea.
Namespaces are one honking great idea -- let's do more of those!

Traducción e interpretación

🎨 Bello es mejor que feo

El código debe ser estéticamente agradable y elegante. La belleza en el código significa claridad y simplicidad.

🔍 Explícito es mejor que implícito

El código debe ser claro sobre lo que hace. No ocultes la funcionalidad detrás de "magia" innecesaria.

✨ Simple es mejor que complejo

Busca la solución más simple que funcione. La simplicidad es una virtud en programación.

📖 La legibilidad cuenta

El código se lee más veces de las que se escribe. Prioriza que sea fácil de entender.

🎯 Debe haber una forma obvia de hacerlo

Python promueve que haya una manera clara y preferida de resolver cada problema.

⏰ Ahora es mejor que nunca

Es mejor hacer algo ahora que posponerlo indefinidamente, pero no a costa de la calidad.

Ejemplos prácticos

Veamos cómo aplicar algunos principios del Zen en código real:

❌ Código que viola el Zen 
# Implícito y poco claro
def f(x):
    return x[0] + x[1] if len(x) > 1 else x[0]

# Complejo e ilegible
resultado = sum([i for i in range(100) if i % 3 == 0 or i % 5 == 0])

# Variables con nombres poco descriptivos
a = 10
b = 20
c = a * b
✅ Código que sigue el Zen 
# Explícito y claro
def sumar_primeros_elementos(lista):
    """Suma los primeros dos elementos de una lista, o el primero si solo hay uno."""
    if len(lista) > 1:
        return lista[0] + lista[1]
    else:
        return lista[0]

# Simple y legible
numeros_especiales = []
for numero in range(100):
    if numero % 3 == 0 or numero % 5 == 0:
        numeros_especiales.append(numero)
resultado = sum(numeros_especiales)

# Variables con nombres descriptivos
base = 10
altura = 20
area_rectangulo = base * altura
🎓 Reflexión: El Zen de Python no son reglas estrictas, sino guías filosóficas. A medida que programes más, entenderás mejor cómo aplicarlas. La experiencia te enseñará cuándo seguirlas al pie de la letra y cuándo ser pragmático.

¿Por qué es importante el Zen?

  • Cultura de la comunidad: Une a los programadores de Python bajo una filosofía común
  • Calidad del código: Fomenta buenas prácticas de programación desde el inicio
  • Mantenibilidad: El código que sigue el Zen es más fácil de mantener y entender
  • Colaboración: Facilita el trabajo en equipo al tener estándares compartidos
  • Aprendizaje: Guía a los principiantes hacia el "estilo Python"

Entrada y Salida de Datos

Función print() - Mostrar información

La función print() permite mostrar información en la pantalla. Es la forma más común de ver resultados en Python.

Ejemplos de print() 
# Diferentes formas de usar print()

# Imprimir texto simple
print("Hola desde Python")

# Imprimir múltiples valores
print("El resultado es:", 42)

# Imprimir con separadores personalizados
print("Python", "es", "genial", sep=" → ")

# Concatenación de strings
nombre = "Ana"
edad = 20
print("Me llamo " + nombre + " y tengo " + str(edad) + " años")

# f-strings (forma moderna y recomendada)
print(f"Me llamo {nombre} y tengo {edad} años")

# Saltos de línea
print("Primera línea\nSegunda línea\nTercera línea")

Función input() - Recibir datos del usuario

La función input() permite que el usuario ingrese información desde el teclado.

⚠️ Importante: input() siempre devuelve un string (texto). Si necesitas números, debes convertirlos usando int() o float().
Ejemplos de input() 
# Ejemplo 1: Entrada de texto
nombre = input("¿Cómo te llamas? ")
print(f"¡Hola, {nombre}!")

# Ejemplo 2: Entrada de números
edad_texto = input("¿Cuántos años tienes? ")
edad = int(edad_texto)  # Convertir string a entero
print(f"Tienes {edad} años")
print(f"El próximo año tendrás {edad + 1} años")

# Ejemplo 3: Operaciones matemáticas
num1 = int(input("Primer número: "))
num2 = int(input("Segundo número: "))

suma = num1 + num2
resta = num1 - num2
producto = num1 * num2

print(f"\n{num1} + {num2} = {suma}")
print(f"{num1} - {num2} = {resta}")
print(f"{num1} × {num2} = {producto}")

Estructuras Condicionales - IF

¿Qué son las condicionales?

Las estructuras condicionales permiten que tu programa tome decisiones. Ejecutan diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

Sintaxis básica:
if condición:
    # código si es verdadero
else:
    # código si es falso

IF simple y IF-ELSE

Ejemplos básicos 
# Ejemplo 1: IF simple
edad = 18

if edad >= 18:
    print("✓ Eres mayor de edad")
    print("Puedes votar")

print("Este mensaje siempre se muestra")

# Ejemplo 2: IF-ELSE
temperatura = 25

if temperatura > 30:
    print("🌡️ Hace mucho calor")
else:
    print("🌡️ El clima está agradable")

# Ejemplo 3: Operadores de comparación
numero = 42

if numero == 42:
    print(f"✓ {numero} es igual a 42")

if numero != 100:
    print(f"✓ {numero} es diferente de 100")

if numero > 40:
    print(f"✓ {numero} es mayor que 40")

if numero <= 50:
    print(f"✓ {numero} es menor o igual a 50")

IF-ELIF-ELSE (múltiples condiciones)

Sistema de calificaciones 
# Sistema de calificaciones
nota = 85

if nota >= 90:
    print("📝 Calificación: A - ¡Excelente!")
elif nota >= 80:
    print("📝 Calificación: B - ¡Muy bien!")
elif nota >= 70:
    print("📝 Calificación: C - Bien")
elif nota >= 60:
    print("📝 Calificación: D - Suficiente")
else:
    print("📝 Calificación: F - Necesitas mejorar")

# Condiciones con operadores lógicos
edad = 20
tiene_licencia = True

print("\n--- Verificación para conducir ---")
if edad >= 18 and tiene_licencia:
    print("✓ Puedes conducir")
elif edad >= 18 and not tiene_licencia:
    print("⚠️ Tienes la edad pero necesitas licencia")
else:
    print("✗ No tienes la edad suficiente")

# Verificar rangos
temperatura = 22

print("\n--- Estado del clima ---")
if temperatura < 0:
    print("❄️ Congelamiento")
elif 0 <= temperatura < 15:
    print("🧊 Frío")
elif 15 <= temperatura < 25:
    print("😊 Templado")
elif 25 <= temperatura < 35:
    print("☀️ Calor")
else:
    print("🔥 Extremadamente caliente")

Operadores en Python

Operadores de comparación:

  • == : Igual a
  • != : Diferente de
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual que
  • <= : Menor o igual que

Operadores lógicos:

  • and : Y lógico (ambas condiciones deben ser verdaderas)
  • or : O lógico (al menos una condición debe ser verdadera)
  • not : Negación (invierte el valor de verdad)

Bucles FOR

¿Qué es un bucle FOR?

Un bucle for permite repetir un bloque de código un número específico de veces. Es perfecto para iterar sobre secuencias como listas, rangos de números, strings, etc.

Sintaxis básica:
for variable in secuencia:
    # código a repetir

FOR con range()

La función range() genera una secuencia de números. Es la forma más común de usar bucles for.

Ejemplos con range() 
# Ejemplo 1: range(n) - de 0 a n-1
print("Contando del 0 al 4:")
for i in range(5):
    print(f"  {i}")

# Ejemplo 2: range(inicio, fin) - de inicio a fin-1
print("\nContando del 1 al 10:")
for i in range(1, 11):
    print(i, end=" ")
print()  # Salto de línea

# Ejemplo 3: range(inicio, fin, paso)
print("\nNúmeros pares del 0 al 20:")
for i in range(0, 21, 2):
    print(i, end=" ")
print()

# Ejemplo 4: Conteo regresivo
print("\n🚀 Cuenta regresiva:")
for i in range(5, 0, -1):
    print(f"  {i}...")
print("  ¡Despegue! 🚀")

# Ejemplo 5: Tabla de multiplicar
numero = 7
print(f"\nTabla del {numero}:")
for i in range(1, 11):
    resultado = numero * i
    print(f"  {numero} × {i} = {resultado}")

FOR con listas y strings

Iteración sobre secuencias 
# Ejemplo 1: Iterar sobre una lista
frutas = ["🍎 Manzana", "🍌 Banana", "🍊 Naranja", "🍇 Uvas"]
print("Mi lista de frutas:")
for fruta in frutas:
    print(f"  - {fruta}")

# Ejemplo 2: Iterar sobre un string
palabra = "PYTHON"
print(f"\nLetras de '{palabra}':")
for letra in palabra:
    print(f"  [{letra}]", end=" ")
print()

# Ejemplo 3: Suma acumulada
numeros = [10, 20, 30, 40, 50]
suma = 0
print(f"\nSumando números: {numeros}")
for num in numeros:
    suma += num
    print(f"  Suma parcial: {suma}")
print(f"Suma total: {suma}")

# Ejemplo 4: FOR con condicionales
print("\nNúmeros del 1 al 15 (FizzBuzz):")
for i in range(1, 16):
    if i % 3 == 0 and i % 5 == 0:
        print(f"  {i}: FizzBuzz")
    elif i % 3 == 0:
        print(f"  {i}: Fizz")
    elif i % 5 == 0:
        print(f"  {i}: Buzz")
    else:
        print(f"  {i}")

# Ejemplo 5: Patrón con FOR
print("\nPatrón de asteriscos:")
for i in range(1, 6):
    print("  " + "★" * i)

Funciones útiles con FOR

  • range(n) : Genera números de 0 a n-1
  • range(inicio, fin) : Genera números de inicio a fin-1
  • range(inicio, fin, paso) : Genera números con incremento personalizado
  • len(lista) : Devuelve el tamaño de una lista
  • enumerate(lista) : Itera con índice y valor
